Fire Prediction for Greece

Scope

The scope of this project is to create a near real-time model that predicts the risk of wildfire occurences in Greek area are before they widespread. The outcome of this prediction is classified and presented in a dynamic map on a web application.

FAQs

How was it implemented ?

  1. Collect past data for fire incidents in Greece as well as temperature, wind speed and dew point
  2. Create a Machine Learning model based on the collected data
  3. Scrape the current teperature, wind speed and dew point of each area to feed my model and classify the area according to its risk level
  4. Fetch the information on a dynamic map

What tools did I used ?

Implementation code was written in Python 3 using the following libraries:

  • Numpy and Pandas: For data processing
  • Sklearn, Tensorflow, Keras: For implementing the various machine learning algorithms
  • Folium: To visualize the results on a dynamic map
  • BeautifulSoup and Requests: For the web scraping process

Details about the project

The aim of this study is to exploit existing free to use data in order to train machine learning models to predict the risk level of fire incidents Our hypotheses is that by collecting historical weather data as well as fire incidents from the web, we can combine them and use them to create machine learning models The model can then be used in real-time to provide a risk level of fire

How it works?

Based on the machine learning model that we have already created, we collect the current teperature, wind speed and dew point of each area and predict the risk level of a fire incident.
